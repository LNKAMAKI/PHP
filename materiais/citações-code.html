CITAÇÕES E CÓDIGOS

TRECHOS DE CÓDIGO

O compartilhamento de código entre desenvolvedores e programadores é muito comum, por isso, existe uma tag para indicar esse tipo de texto: a tag code.

code
function () {
console.log('hey')
}
code

Colocando esse código no documento HTMl, o resultado na tela será o seguinte:

function () { console.log('hey') }

Isso porque a tag code serve somente para indicar um trecho de código, ou seja, possui um valor semântico, e deixa a fonte monoespaçada; ela não deixa o código formatado do jeito que nós o escrevemos. Se quisermos fazer isso, utilizamos a tag pre:

pre
code
function () {
console.log('hey')
}
code
pre
