CITAÇÕES E CÓDIGOS

TRECHOS DE CÓDIGO

O compartilhamento de código entre desenvolvedores e programadores é muito comum, por isso, existe uma tag para indicar esse tipo de texto: a tag code.

code
function print() {
console.log('hey')
}
code

Colocando esse código no documento HTMl, o resultado na tela será o seguinte:

function print() { console.log('hey') }

Isso porque a tag code serve somente para indicar um trecho de código, ou seja, possui um valor semântico, e deixa a fonte monoespaçada; ela não deixa o código formatado do jeito que nós o escrevemos. Se quisermos fazer isso, utilizamos a tag pre:

pre
code
function print() {
console.log('hey')
}
code
pre
 CITAÇÕES

Também é comum os sites possuírem trechos de citações retirados de outras referências. É justamente para isso que existem a tag q e blockquote:

q
Tell me and I forget, Teach me and I remember, Involve me and I learn.
q
Tell me and I forget, Teach me and I remember, Involve me and I learn.

blockquote
Lorem ipsum dolor sit amet consectetur adipisicing elit. Quod eos quo error quos ea sint consequatur ipsum quis soluta. Autem ut sed saepe, explicabo minima aspernatur et ipsa deserunt.
blockquote
Lorem ipsum dolor sit amet consectetur adipisicing elit. Quod eos quo error quos ea sint consequatur ipsum quis soluta. Autem ut sed saepe, explicabo minima aspernatur et ipsa deserunt.

Ainda é possível colocar o parâmetro cite dentro da tag blockquote, que recebe o link para o site de onde foi retirado.
