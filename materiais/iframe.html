COLOCANDO UM IFRAME

E como colocar um iframe? Dê uma olhada no código abaixo para descobrir:

iframe scr="site.html" frameborder="0"iframe

Aí agora você deve estar se perguntando: o que é esse frameborder? E aí eu te respondo: um parâmetro bem inútil se quiser saber a verdade. Brincadeirinha (se bem que é isso mesmo). O frameborder é basicamente a borda do iframe, mas ele só admite os valores 0 e 1, o que o torna não muito eficiente.

Assim como para áudio ou vídeo, é possível colocar um parágrafo ou link (ou outro elemento) dentro do iframe, para caso ele não seja compatível com o navegador:

iframe scr="site.html" frameborder="0"
a href="site.html"Acesse aqui o sitea
iframe

Ainda tem outros parâmetros, não muito úteis em minha opinião, mas que é bom saber: o width, o height e o scolling.

Assim como no CSS, é possível configurar a largura e o comprimento do iframe no próprio HTML, com o width e o height:

iframe scr="site.html" width="100" height="200"iframe

O scolling é também um tanto inútil. Ele se refere a rolagem do iframe e pode receber 3 valores: yes (que cria a rolagem mesmo que o conteúdo caiba no iframe), no (que não cria a rolagem) e auto (que cria a rolagem quando o conteúdo excede o iframe).
