LISTAS OL

As listas ordenadas são aquelas em que há uma contagem do primeiro ao último itens contidos nelas:

  1. Melancia
  2. Maça
  3. Banana
  4. Manga
  5. Laranja
  6. Morango

O tipo dessa lista de cima é "1", pois começa no número 1.

  1. Melancia
  2. Maça
  3. Banana
  4. Manga
  5. Laranja
  6. Morango

Já o tipo dessa é "a"

  1. Melancia
  2. Maça
  3. Banana
  4. Manga
  5. Laranja
  6. Morango

"A"

  1. Melancia
  2. Maça
  3. Banana
  4. Manga
  5. Laranja
  6. Morango

"i"

  1. Melancia
  2. Maça
  3. Banana
  4. Manga
  5. Laranja
  6. Morango

"I"

Cada item dessas listas está contido em uma tag li, de list item, que pode ou não possuir fechamento (opcional).

Veja o código para criar uma lista ordenada simples:

ol
li ITEM 1 li
li ITEM 2 li
li ITEM 3 li
ol
