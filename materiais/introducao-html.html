HYPERTEXT MARKUP LANGUAGE

HTMl significa Hypertext Markup Language, ou seja, não é uma linguagem de programação e, sim, de marcação. Ela faz parte do conjunto das 3 linguagens que geralmente fazem parte de um site: HTML, CSS e JS(Javascript).

Essa linguagem pode ser considerada a base da construção de um site, pois ela serve para criarmos o conteúdo que fará parte dele(vídeos, imagens, áudios, textos, tabelas, etc).

HTML faz uso de tags ou etiquetas para criar os conteúdos. Para fazer um parágrafo, por exemplo, utilizamos a tag p:

pEstrutura de uma tagp

Portanto, para abrir uma tag colocamos o seu nome dentro de colchetes angulares (<...>) e a fechamos usando colocando / na frente (</...>).

Podemos, assim, colocar texto ou outras tags dentro de uma tag.

ESTRUTURA HTML BÁSICA

Primeiramente, para criar um site precisamos gerar a estrutura a seguir, digitando ! e dando Enter na primeira opção (referente à HTML-5, a versão mais recente).

!DOCTYPE html
html lang="en"
head
meta charset="UTF-8"
meta http-equiv="X-UA-Compatible" content="IE=edge"
meta name="viewport" content="width=width-device-width, initial-scale=1.0"
titleDocumenttitle
head
body

body
html

Explicar o que cada parte significa não é necessário. O mais importante é saber que podemos mudar o parâmetro lang, atribuída à tag html e o título do documento. A língua padrão é inglês, mas podemos mudar para português brasileiro, colocando "pt-br" no lugar de "en".

O body é onde o conteúdo irá ficar. Então se quisermos, por exemplo, colocar um cabeçalho principal, podemos adicionar a tag h1 dentro do body:

Só por curiosidade mesmo, mas há uma "superstição" acerca do "Olá mundo", razão por qual é comum vermos essa frase em programação ou coisa do tipo. Segundo essa "superstição", se você não escrever "Olá Mundo" no seu primeiro projeto, digamos assim, então você não será bem-sucedido na área de computação.

H1, H2, H3, H4, H5, H6

Falando em h1, vamos dar uma olhada nos outros 5 h's (headers) que existem. O que difere um do outro visualmente é o tamanho da fonte - o h1 é maior que o h2, o h2 maior que o h3 e assim por diante:

Porém, eles também têm valores semânticos, ou seja, significados diferentes, já que o h1, por ser maior, está em um nível superior que os demais.

Com isso, concluímos que as tags em HTML podem ter tanto valores semânticos como visuais.
