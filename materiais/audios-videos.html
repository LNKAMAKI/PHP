ÁUDIOS E VÍDEOS

ÁUDIOS

Vamos agora falar sobre como colocar áudios em nosso site. Lembrando que, como qualquer outro tipo de mídia, áudios possuem direitos autorais e, por isso, não podemos pegar qualquer áudio e usá-lo sem permissão.

O exemplo abaixo utiliza a tag audio e já vem com os controles para rodá-lo.

O código usado foi o abaixo e já é o suficiente para você inserir um áudio em seu site:

audio scr="audio.mp3" controlsaudio

Viu que fácil? Mas não é só isso. Também é possível fazer com que o áudio se repita infinitamente, usando o loop, que ele seja rodado ssim que a página for carregada, com o autoplay, e que seja carregado previamente (preload="metadata").

audio scr="audio.mp3" loop autoplay preload="metadata"audio

Por último, é importante saber que nem todos os navegadores são compatíveis a determinado formato de áudio. Assim, o recomendado é ter o maior número de formatos para o seu áudio, para que não haja esse problema.

Podemos fazer isso colocando as tags sources dentro da tag audio:

audio
source scr="audio.mp3" type="audio/mpeg"
source scr="audio.ogg" type="audio/ogg"
source scr="audio.wav" type="audio/wav"
pNão foi possível carregar o áudio :(p
audio

Como você pode ver no código, também é possível colocar um parágrafo para quando nenhum dos áudios for compatível e, ainda, colocar dentro dele um link para baixar o áudio.

VÍDEOS

Agora vamos falar sobre vídeos. A forma de inserir vídeos em um site é semelhante à de audios. Veja abaixo um exemplo:

video scr="video.mp4" controls autoplayvideo

O problema da falta de compatibilidade também se aplica aos vídeos, podendo ser resolvido pela maneira:

video
source scr="video.mp4" type="video/mp4"
source scr="video.m4v" type="video/mp4"
source scr="video.webm" type="video/webm"
source scr="video.ogv" type="video/ogg"
pNão foi possível carregar o vídeo :(p
